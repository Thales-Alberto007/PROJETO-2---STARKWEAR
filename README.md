
Nosso site foi criado a partir de uma ideia recorrente devido a um primeiro projeto de HTML e CSS já existente.Com isso, o grupo decidiu a junção da ideia desse projeto com uma atualizçaão no visual do site e  a inserção do backend juntamente com o banco de dados. Dito isso, Starkwear é uma loja virutal de camisas estilosas e estampas únicas. Conheça nosso ambiente virtual com segurança e confiabilidade. Nosso site conta com um desing simples e interativo, tudo criado foi pensado pela equipe a fim de garantir acessibilidade e simplicidade na navegação. Contamos com uma página inicial, uma aba de "fale conosco", "sobre a loja", "quem somos" e outra páginas feitas a fim de estabelecer um contato mais próximo com nossos clientes.
